## 📝 Sobre los Informes

Cada informe está redactado de forma profesional, ya sea en formato técnico o ejecutivo:

- **Informe Técnico:** detalle del proceso de enumeración, explotación y post-explotación.
- **Informe Ejecutivo:** resumen orientado a un público no técnico, con hallazgos y recomendaciones.

Los documentos están escritos en **LaTeX** para garantizar un formato limpio y profesional, y exportados como **PDF**.
